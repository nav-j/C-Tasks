## Task: Check Whether a Number is Even or Odd

###  Problem Statement:

Write a C program that:

1. Declares an integer variable and assigns it a value.
2. Uses an `if else` statement to check whether the number is **even** or **odd**.
3. If the number is even, print:
   `"The number is even."`
4. If the number is odd, print:
   `"The number is odd."`

---

###  Example:

If the number is `10`, the output should be:

```
The number is even.
```
