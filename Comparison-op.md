## **Task: Comparison Operators in C**

> **Note:**
> Comparison operators return `1` (true) or `0` (false).

---

### **Task 1: Greater Than (`>`)**

```c
int main() {
    int a = 15;
    int b = 10;
    printf("%d", a > b);
    return 0;
}
```

---

### **Task 2: Less Than (`<`)**

```c
int main() {
    int x = 7;
    int y = 12;
    printf("%d", x < y);
    return 0;
}
```

---

### **Task 3: Equal To (`==`)**

```c
int main() {
    int p = 9;
    int q = 9;
    printf("%d", p == q);
    return 0;
}
```

---

### **Task 4: Not Equal To (`!=`)**

```c
int main() {
    int a = 20;
    int b = 25;
    printf("%d", a != b);
    return 0;
}
```

---

### **Task 5: Greater Than or Equal To (`>=`)**

```c
int main() {
    int x = 10;
    int y = 10;
    printf("%d", x >= y);
    return 0;
}
```

---

### **Task 6: Less Than or Equal To (`<=`)**

```c
int main() {
    int m = 6;
    int n = 4;
    printf("%d", m <= n);
    return 0;
}
```

---

### **Task 7: Comparison with Arithmetic Expression**

```c
int main() {
    int a = 12;
    int b = 5;
    printf("%d", a / b > 2);
    return 0;
}
```

---

### **Task 8: Multiple Comparisons**

```c
int main() {
    int x = 8;
    int y = 3;
    printf("%d", x > y && x != 10);
    return 0;
}
```
