
##  **Task: Arithmetic Operators in C**

### **Task 1: Addition & Subtraction**

```c
int main() {
    int a = 18;
    int b = 7;
    int c = a - b + 5;
    printf("%d", c);
    return 0;
}
```

---

### **Task 2: Multiplication & Division**

```c
int main() {
    int x = 24;
    int y = 5;
    int z = x * y / 10;
    printf("%d", z);
    return 0;
}
```

---

### **Task 3: Modulus Operator**

```c
int main() {
    int p = 35;
    int q = 8;
    printf("%d", p % q);
    return 0;
}
```

---

### **Task 4: Mixed Arithmetic Operators**

```c
int main() {
    int a = 10;
    int b = 4;
    int c = a + b * 3 - a / b;
    printf("%d", c);
    return 0;
}
```

---

### **Task 5: Increment Operator**

```c
int main() {
    int x = 6;
    x++;
    x = x * 3;
    printf("%d", x);
    return 0;
}
```

---

### **Task 6: Decrement Operator**

```c
int main() {
    int y = 20;
    y--;
    y = y / 3;
    printf("%d", y);
    return 0;
}
```

---

### **Task 7: Operator Precedence**

```c
int main() {
    int a = 8;
    int b = 2;
    int c = a * b + a / b - b;
    printf("%d", c);
    return 0;
}
```
