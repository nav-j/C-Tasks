## **Task 1: Predict the Output**

Study the following C program and write the output of each `printf()` statement.

```c
#include <stdio.h>

int main() 
{
    int a = 10, b = 3, c;

    c = a + b * 2;
    printf("1. %d\n", c);

    c = (a + b) * 2;
    printf("2. %d\n", c);

    c = a % b;
    printf("3. %d\n", c);

    c = a++;
    printf("4. %d\n", c);

    printf("5. %d\n", a);

    c = ++b;
    printf("6. %d\n", c);

    printf("7. %d\n", (a > b));

    printf("8. %d\n", (a > b && b < 5));

    printf("9. %d\n", (a == b || b == 4));

    return 0;
}
```
